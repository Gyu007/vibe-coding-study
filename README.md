# 나의 사이트

바이브 코딩 스터디 과제로 만든 간단한 다중 페이지 HTML 사이트입니다. 별도의 프레임워크나 빌드 도구 없이 순수 HTML/CSS로 작성되어 브라우저에서 바로 열어볼 수 있습니다.

## 구성

- `index.html` - 홈 페이지
- `about.html` - 소개 페이지
- `contact.html` - 연락처 페이지 (전송 기능 없는 폼)
- `css/style.css` - 공통 스타일시트
- `images/` - 이미지 리소스

## 실행 방법

별도의 설치 과정 없이 `index.html` 파일을 브라우저에서 열면 됩니다.

```bash
# 예: 기본 브라우저로 열기 (Windows)
start index.html
```
